# Lumina-AI
# AI-Powered Micro-SaaS Customer Support Bot

A production-ready Streamlit application that lets you create, train, and deploy AI customer support bots using OpenAI and Retrieval-Augmented Generation (RAG).

## Features
- **User Authentication** – Secure signup/login with bcrypt.
- **Bot Builder** – Define persona, upload PDF/TXT knowledge bases, and train a vector store.
- **Live Chat Playground** – Test your bot in a beautiful chat UI.
- **Integration Simulator** – Generate webhook URLs and simulate incoming messages (WhatsApp/Shopify-style).
- **Analytics Dashboard** – Quick stats on bots and conversations.

## Tech Stack
- **Frontend**: Streamlit + Custom CSS (glassmorphism)
- **AI/LLM**: LangChain, OpenAI, FAISS
- **Database**: SQLite
- **Deployment**: Streamlit Sharing / Cloud
